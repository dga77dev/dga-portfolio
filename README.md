# CV + Portfolio

Este es mi curriculum vitae + portfolio creado con REACT + Vite.

En App.jsx se encuentran todas las variables que dan contenido al curriculum. Donde se añade el contenido (formación, experiencia, etc) sin tener que tocar el código.

Eres libre de descargarlo, modificarlo y utilizarlo. No eres libre de utilizar imagenes y datos personales.

Puedes ver CV [aquí](https://dga-portfolio.web.app).
